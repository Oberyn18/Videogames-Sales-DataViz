Vendas de Videogames

As quatro principais regiões de vendas videogames no mundo

O mapa contem o mundo inteiro dividido em 4 regiões, onde foram feitas vendas de videogames. Cada região tem uma cor que o diferença do resto.

Você pode aumentar ou diminuir o zoom, ou arrastar o mapa para encontrar uma região ou país específico. Também pode fazer clic para selecionar uma região e mudar os dados das outras representações.

Evolução das vendas de videogames em cada região ao longo dos anos

Este gráfico de fluxo (streamgraph) mostra as vendas de videogames através dos anos para cada região, a altura de cada fluxo indica a quantidade de vendas. Se alguma das regioes do mapa está selecionada, esta visualização mostrará as plataformas para as quais os videogames foram vendidos ao longo dos anos.

Você pode pasar o mouse por acima de algum fluxo para olhar a quantidade de vendas para uma plataforma específica em um ano específico.

Gêneros de videogames vendidos em cada regiao

O gráfico de barras mostra as vendas de videogames categorizadas por gêneros. Se alguma das regiões está selecionada no mapa, uma filtragem é aplicada sobre o gráfico para mostrar somente a informacao da região selecionada.

Você pode pasar o mouse por acima de alguma barra para olhar o número exato de vendas do gênero de videogame correspondente.

Vendas de videogames nas quatro regiões ao longo dos anos

O gráfico de série mostra as vendas de cada região através dos anos. Se alguma das regioes está selecionada no mapa, a linha da regiao correspondente neste gráfico é também selecionada.
